#WML

Machine learning tools library.

##Dependencies

- Linux or Mac OS
- Python ≥ 3.6
- scipy
- yacs
- OpenCV
- pycocotools
- gcc & g++ ≥ 4.9

##Installation

```
pip install -r requirements.txt
```
